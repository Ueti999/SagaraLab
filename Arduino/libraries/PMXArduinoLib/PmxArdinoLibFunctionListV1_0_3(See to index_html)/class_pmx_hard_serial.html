詳解

PMXで使用する関数や定義をひとまとまりにしたもので、ArduinoのHardwareSerialを使用して動作します。

・PMXの固有値の定義 ・PMXのコマンドの生成、チェックおよび送受信(送信部分は外部依存) ・PMXで必要だと思われる機能の関数化 ・通信部分はArduinoのHardwareSerialを使用しています
