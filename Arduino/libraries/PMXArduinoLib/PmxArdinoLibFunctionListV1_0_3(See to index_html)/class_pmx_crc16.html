詳解

PMXで使用するCRCの演算

  • 生成多項式：x^16+x^12+x^5+1 (CRC-16-CCITT)
  • ビットシフト方向：左送り
  • 初期値：0x0000
  • 出力：非反転 （出力XOR：0x0000）
