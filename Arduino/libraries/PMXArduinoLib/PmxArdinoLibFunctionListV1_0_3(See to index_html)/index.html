PmxArdinoLibの概要

このライブラリは近藤科学製PMXサーボモータと通信するための関数になります
使い方および詳細は、下記弊社HPをご覧ください。
http://kondo-robot.com/
不具合等ありましたら、弊社HPを参照にご連絡ください。
PmxBaseは、PMXの基本となる部分を記載しています。 通信の違いを吸収するためHardwareSerialやSoftwareSerialに派生しています。
